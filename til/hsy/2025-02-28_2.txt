4. 적대적 생성 신경망
1) 적대적 생성 신경망 (GAN, Generative Adversarial Network)
- 두 개의 신경망(생성자 + 판별자)이 경쟁하며 데이터를 생성하는 모델

<GAN 기본 구조> 
- Generator (생성자, G)
: 랜덤 노이즈를 입력으로 받아 실제 데이터와 비슷한 데이터를 생성
: 판별자를 속일 만큼 진짜 같은 데이터 생성이 목표

- Discriminator (판별자, D)
: 실제 데이터와 생성된 데이터를 입력으로 받아 진짜/가짜를 구별
: 생성자가 만든 가짜 데이터를 정확히 판별하는 것이 목표

- 학습 방식 (적대적 훈련, Adversarial Training)
: 생성자는 더 진짜 같은 데이터를 만들고, 판별자는 더 정확하게 구별하도록 경쟁
: 최종적으로 판별자가 생성자의 데이터읠 구별할 수 없을 정도가 되면 학습이 성공한 것

<GAN의 목적 함수>
GAN은 Two-Player Zero-Sum Game(두 플레이어의 적대적 게임)구조
- 판별자(D): 가짜 데이터를 정확히 판별하려고 함 -> 손실을 최대화
- 생성자(G): 판별자를 속이려고 함 -> 손실을 최소화
=> GAN은 생성 모델이 판별자를 속일 수 있도록 최적화되는 과정

2) 조건부 생성 모델 (cGAN, Conditional GAN)
- 단순히 무작위 데이터를 생성하는 것이 아니라, 특정한 조건을 주고 원하는 데이터를 생성하는 모델

<기존 GAN의 문제점>
GAN은 데이터를 생성할 수 있지만 어떤 데이터가 생성될지 조절할 수 없음

<cGAN의 구조>
기존 GAN과 비슷하지만 생성자(G)와 판별자(D) 모두 조건 c를 추가로 입력받음
=> GAN은 단순한 생성 모델을 넘어, 원하는 데이터를 조절할 수 있는 모델
